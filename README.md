## 使用 Usage

- 拖拽批量压缩图片, 支持格式 `jpg/png/gif`
> Drop to optimize, `jpg/png/gif` is supported.

- 拖拽批量上传图片到对象存储, 并默认开启图片压缩
> Drop to upload to Cloud Object Storage Server, and optimize by default.

## 支持 Supoorted
- 阿里云 OSS
- 腾讯云 COS
- 七牛云 Qiniu
- Github
