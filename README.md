## 使用 Usage

- 拖拽批量压缩图片, 支持格式 `jpg/png/gif`
> Drop to optimize, `jpg/png/gif` is supported.

- 拖拽批量上传图片到 阿里云OSS/七牛云, 并默认开启图片压缩
> Drop to upload to alioss/qiniu Object Storage, and optimize by default.
